# 김영한 자바 입문
## 조건문
- 조건이 참(true)일 때 코드블럭 실행

### if문
- `if`
- `if else`
- `if else if`
  - if문 묶어서 사용
  - 전체 중에 하나만 실행돼
  - 조건이 없으면 다음조건...
  - 조건들이 체크된 상태로 다음 조건으로 넘어가
  - 조건 체크 중복 피할 수
- if문 {} 중괄호 그냥 써라. 가독성

### switch문
- 등급에 따른 로직 실행.
  - 조건식 넣는 부분에 단순 값만
  - `if`문은 조건식이 더 자유로움
- `break` 로 case 탈출

### 자바 14의 새로운 switch문
if문보다 더 깔끔한 switch문   
아직 많이 사용하지 않음.
- 기존 `switch`문과 차이
  - `->`를 사용.
  - 선택된 데이터를 반환

### 삼항 연산자
- 조건에 따라 값을 대입하고 싶을 때
